# graphviz 描述语言

> 2023 11月25日

## 介绍一下

> https://graphviz.org/

graphviz是一个开源工具包，用于绘制“DOT”语言描述的图形。dot语言是图形描述语言，以`.dot`结尾，其简单的语法和文本字符形式，便于记录、分发、以及后续修改。在编写好后，graphviz负责将它渲染成png、pdf、或svg，其中，图形会自动进行排布。
